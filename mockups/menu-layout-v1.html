Layout rules this wireframe encodes

  • Four zones, four containers. .topbar, .leftrail, .rightrail, .timeline. Every floating control is a child of one of these — no more loose position:fixed islands with magic rem offsets.
  • One button vocabulary. .ctrl-btn replaces .view-btn, .search-pill, and .fab. Variants: .ctrl-btn--icon (square, icon only) and .ctrl-btn--rail (full-width inside the left rail). Same height (36px desktop / 44px mobile), same radius, same glass hover, same accent active state.
  • Top bar. Small wordmark left · breadcrumb center (with folded Back/Home arrows as prefix) · lang/theme/profile right. Subtitle dropped — splash only.
  • Left rail. Flex-column glass panel, bounded by top:72px/bottom:120px so it can never collide with top bar or timeline. Order: View modes → divider → Tools (Compare, Games, Stories, Hominins) → divider → Extinct + Tour → foot: merged status badge + Species of the Day.
  • Right rail. Zoom only. Vertically centered. Narrow (36px column + padding).
  • Search. Icon-only by default, sits just to the right of the left rail on desktop. Click expands a 280px input inline. On mobile the icon lives top-left next to the hamburger and expands into a full-width overlay.
  • Timeline. Single 88px band at the bottom: one info row (era label · visible count · play/speed buttons) + the color-coded track with extinction ticks + tiny era axis labels. The old "era presets" row and separate extinction shortcut row are gone — extinctions collapse into tick marks on the track itself.
  • Status. One merged badge ("🔬 Seen N / M species") in the left rail foot. Old #progress-badge + species-count strings deduped.
  • Legend removed entirely. Footer + shortcuts hint collapsed into one 10px corner credit.
  • Mobile. Left rail collapses behind a ☰ button top-left; tapping it slides a bottom-sheet up containing the same View / Tools / Extinct / Tour groups in a 3-column grid (44px touch targets). Right rail shrinks to 3 zoom buttons and sits just above the timeline. Breadcrumb hidden (already navigable via panel).

Duplicates removed

  • #btn-trivia FAB — killed. Games lives only in left-rail Tools.
  • #btn-compare FAB — killed. Compare lives only in left-rail Tools.
  • #btn-quiz pill — killed. Same Games entry point.
  • #legend — deleted.
  • #progress-badge + "N species visible" caption — merged into one status badge.
  • #footer + #shortcuts-hint — collapsed into the single corner credit.

New CSS tokens

  • --ctrl-h: 36px / --ctrl-h-mobile: 44px
  • --rail-w-expanded: 188px (left rail content width)
  • --glass-bg, --glass-border, --glass-blur — single source for every frosted surface

What breaks vs. current code

  • HTML structure moves: #view-toggle leaves #top-right-controls and becomes a child of the new #left-rail. IDs on individual buttons stay stable so JS onclick handlers keep working.
  • #search-wrap's #search-pill-row is dismantled — each pill relocates: Extinct → left rail, Hominins/Games/Stories → left rail Tools.
  • #nav-ctrl Back/Home move inside #breadcrumb as prefix buttons.
  • .fab-group (whole block) deleted.
  • #legend deleted.
  • #timeline's #era-presets row removed (presets → optional dropdown later if needed). Extinction markers remain as on-track ticks.
